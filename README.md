# Azamjon IELTS Prep

A free, open-source IELTS preparation application offering practice tests, scoring, and personalized study plans.

## Features

- **3 Complete IELTS Tests**: Practice with full-length tests covering Listening, Reading, Writing, and Speaking.
- **Instant Scoring**: Get detailed feedback and scores immediately after each test.
- **Personalized Study Plans**: Receive a customized study schedule based on your performance.
- **Progress Tracking**: Your progress is automatically saved on your device (no login required).

## How to Use

You can use the application directly by opening the `index.html` file in your browser. Alternatively, you can host it on a web server.

## Project Structure

- `index.html`: The main HTML file.
- `style.css`: The stylesheet for the application.
- `script.js`: The JavaScript file containing the application logic.
